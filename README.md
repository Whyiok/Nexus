# Nexus
**Nexus** - это игровая платформа, узконаправленная на коммьюнити майнкрафта. Эта платформа нужна для тех, кто хочет поделиться своим творчеством, общаться и дисскусировать.
## 1. Возможности сервиса
  1. Просмотр и добавление дисскусий
  2. Просмотр и добавление постов
  3. Просмотр и добавление комментариев
  4. Просмотр новостей
  5. Регистрация и авторизация с подтверждением эл.почты
  6. Редактирование и удаление своих постов и дискуссий
  7. Система лайков и рейтинга постов
  8. Жалобы
## 2. Роли и их возможности
  ### 1. Обычный пользователь
    1. Все вышеперечисленные возможности
    2. Возможность получения блокировки, статуса модератора и статуса доверенного пользователя
  ### 2.  Модератор
    1. Все возможности обычного пользователя
    2. Удаление и редактирование постов / дискуссий обычных пользователей
    3. Модерация постов / дискуссий
    4. Блокировка и удаление аккаунтов обычных и доверенных пользователей
  ### 3. Доверенный пользователь
    1. Все возможности обычного пользователя
    2. Создание заявки на добавления поста во вкладку новости
    3. Создание заявки на блокировку или удаление аккаунтов
    3. Создание заявки на удаление поста
  ### 4. Администратор
    1. Все вышеперечисленные возможности
